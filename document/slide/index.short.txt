O Reactful framework completamente transforma o React!

Reactful é um framework completo com conceitos inovadores que resolve as principais fraquezas do desenvolvimento com React.

Ele corrige o vazamento de escopo no import de CSS, modularizando o CSS sem necessidade de libs como css-module ou styled-component.

Ele corrige a complexidade do tratamento de estado com reative objects, usando props reativos como local states e dependency injection para global states.

Também introduz diretivas de propriedade para dual data binding, que permite binding minimalista e rotas declarativas.

Também implementa decoradores com suporte a funções, que facilita a codificação de rotas dinâmicas, autorização de rotas, renderização híbrida, seach enginering optmization, entre outros.

Para mais informação, vite tinyurl.com/reactful

The Reactful framework completely transforms React!

Reactful is a full-featured framework with innovative concepts that solves the main react developing flaws.

It fixes the global scope leaking in CSS imports, allowing modularized CSS design.

The state management verbosity is fixed by reactive objects and props binding directives.
